# Ebooks Directory

This directory contains the course PDFs:

1. **sample-chapter-one.pdf** - Free sample chapter for download
2. **live-like-a-winner-full.pdf** - Complete ebook (access after payment)

## File Upload Instructions

To upload PDFs to this directory:
1. Use the Softgen file upload feature in the left sidebar
2. Navigate to `/public/ebooks/`
3. Upload your PDF files with the exact names above

## Usage

- Sample chapter: Available to all users via landing page and sample lesson
- Full ebook: Available only to paid users via their dashboard/resources page
